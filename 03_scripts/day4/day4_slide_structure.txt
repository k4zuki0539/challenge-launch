Day4 スライド構造設計書（セールスパート）
	[BLOCK A] 開始前ウォームアップ + ルール（〜5分）
		Day1〜3と同じルールスライド構成
		追加: Day3の課題提出者数・0→1達成者の紹介スライド
	[BLOCK B] ロードマップ振り返り + 理想の未来（5〜10分）
		企画全体の目的スライド（再掲）
		5Days全体ロードマップ → 「今日が最後の壁」と強調
			壁①②完了済み + 壁③がDay4のテーマ
		理想の未来チャット質問
			「0→100達成した後、あなたは何がしたいですか？」（状態で聞く）
		「今日が5日間で一番重要な日」スライド
	[BLOCK C] Day3復習 → 反転（5〜10分）
		Day3の内容を1〜2行で振り返り
		「でも…ここからが本当の勝負です」（転換スライド）
		チャット質問（梯子落とし前の仕掛け）
			「一人でこれを続けられる自信がある人？」→ ほぼいない → 「それが普通です」
	[BLOCK D] 梯子落とし（15〜20分）
		D-1: 副業失敗の現実提示
			Q: 「副業で失敗した経験がある人？」→ 多数確認 → 「それが現実」
		D-2: 3つの壁（なぜ一人では完走できないか）
			壁1: モチベーション維持できない（2〜3枚）
				「"この方向性で合ってる？" "価格設定はこれでいい？" → 誰にも聞けない → 手が止まる」
			壁2: 方向性が分からず不安（2〜3枚）
				「LPを書いた。でも"これで刺さるか"が分からない」
				「3ヶ月後 → "ローンチ失敗した"が分かる。もう遅い」
			壁3: 時間がない（2〜3枚）
				「今日は疲れた…明日やろう」の連鎖 → 1ヶ月後も何も進んでいない
			まとめ: 「この負のループを一人で抜け出すのは極めて困難」
	[BLOCK E] 「祭りの終わり」感情的ピーク（5〜10分）★最重要
		スライド①: 「たった5日間のセミナーで億万長者になれるか？」
			逆説的に「なれません」と言ってから反転させる
		スライド②: セミナー翌朝の情景描写
			「この数日間は私がいて、仲間がいて、チャットが盛り上がっていたから走れた」
			「6日目の朝、PCを開いた時、チャットは動かない」
			「日常の引力があなたを引き戻しに来る」
			「怖くないですか？」
		スライド③: 「今はどうですか？」
			「正しい知識と環境さえあれば、私にもできるんじゃないか？→ そう思えていませんか？」
		スライド④: 「あなたは消費者じゃない。生産者です」
			「ただやり方が分からなかっただけ」
	[BLOCK F] 根本原因の提示（5分）
		「結果が出ない本当の原因」
			"やり方を知らない"ではなく "環境がない"こと
		「これまでの失敗はあなたのせいじゃない」
			「仕組みがなかっただけ。環境がなかっただけ」
	[BLOCK G] 環境の必要性（例え話 3連発）（5分）
		例え1: ジム入会 → 続く / 家で一人 → 3日で終わる
		例え2: 英会話スクール → 話せる / 参考書のみ → 続かない
		例え3: ダイエットコーチ → 痩せる / 一人 → リバウンド
		→ 「副業も同じ」
		成功に必要な環境の3要素
			1. 定期的なフィードバック（「合ってる？」の答え合わせ）
			2. 軌道修正してくれる存在（間違い方向を早期発見）
			3. モチベーションを保つ仕組み = 仲間
	[BLOCK H] 解決策 → 商品紹介（わどAIスクール）（20〜30分）
		H-1: 商品導入
			「成果にコミットした本物の環境を作ることに成功した」
			「なぜこのスクールを作ったのか」（わどの原体験ストーリー）
		H-2: スクール概要
			わどAIスクール — AI×Xで"商品設計→集客→教育→販売"を一気通貫で実行できる環境
		H-3: スクールの特徴（箇条書き + アイコン）
			カリキュラム / 個別フィードバック / コミュニティ / 無期限サポート
		H-4: コース説明（3コース比較表）
		H-5: カリキュラム詳細（番号付きリスト + 想定価値）
		H-6: サポート体制（個別チャット無制限 / 動画添削 / コミュニティ / オフ会）
		H-7: テンプレート・特典展開（ローンチ関連テンプレートに差し替え）
	[BLOCK I] 実績・受講生の声（10〜15分）
		受講生実績（数値スライド × 複数）
		受講生の声（ツイートスクショ形式 × 多数）
		受講生対談の様子（写真・動画）
		成果報告チャット画面
		ポイント: 量で"これだけ結果が出ている"を見せる（15枚以上）
	[BLOCK J] コース・価格・オファー提示（10〜15分）
		J-1: 価格スライド
			「今回は1対1の個別面談で最適なプランをお伝えする」
		J-2: 今日限定特典（通常パッケージ + 本日限定特典の比較）
		J-3: 「参加しなかった場合」スライド
			時系列グラフ（1ヶ月後/3ヶ月後/6ヶ月後）
		J-4: 審査制の説明
			「今回は審査制。誰でも参加できるわけではない」
			理由: サポートの質を担保するため
		J-5: 「私たちの覚悟」スライド
			「売上を上げられなければ全て私たちの責任」
	[BLOCK K] FAQ + スクールのルール（5〜8分）
		FAQ（Q→A形式）
			Q: 経験が必要では？
			Q: 自分には使えないのでは？
			Q: 時間がなくて…
			Q: お金がギリギリで…
			Q: 分割払いは可能？
			Q: 本当に信頼していい？
		スクールのルール（5つ）
			ルール1: 最低1年続ける
			ルール2: お客様扱いしない
			ルール3: 社長の自覚を持つ
			ルール4: 行動量が全て
			ルール5: 自責思考を持つ
	[BLOCK L] QR提示 + 申し込み手順（5〜10分）
		感情的クロージングの一言「一緒に人生を変えましょう」
		QRスライド: 「あなた専用ローンチ設計セッション（個別面談）はこちらから」
			面談で得られるもの: ①ローンチロードマップ ②30日アクションプラン ③正直な判断
		申し込み手順スライド（3〜4枚・スクショ付き）
			「セミナー終了後は申し込めなくなります」の締め切り提示
	[BLOCK M] Q&Aタイム + クロージング
		QRを再掲示しながらQ&A
		最後にXへの感想ポスト依頼
	Day4特有の感情フロー
		[高揚] Day3で「できそう！」
		[転換] 「でも…本当の勝負はここから」
		[現実直視] 「一人で続けられる自信ある？」→ ほぼいない
		[共感] 3つの壁（失敗はあなたのせいじゃない）
		[恐怖] 「祭りが終わる。6日目の朝、一人になる」
		[希望] 「今は"できるかも"と思えているはず」
		[答え合わせ] 「結果が出ない理由は環境がないだけ」
		[解決策] 「本物の環境がここにある」→ スクール紹介
		[信頼構築] 実績・受講生の声
		[決断] QR提示 → 面談申し込み
	Day4のワーク（今回新規追加）
		梯子落としとセールスの間に挿入
		「あなたのローンチ計画、一人で埋められない穴はどこ？書き出してみてください」
		→ 参加者が自分の言葉で「環境が足りない」と気づく
	前回からの改善点
		「個別相談」→「あなた専用ローンチ設計セッション」にリフレーム
		梯子落とし→即セールス → 梯子落とし→ワーク→セールス（自分で気づく設計）
		QR前のボルテージ不足 → 感情曲線を設計し最高潮でQR提示
