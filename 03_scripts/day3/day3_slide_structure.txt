Day3 スライド構造設計書
	[BLOCK A] アウトプット紹介（4〜5枚）
		A-1: 冒頭チャット確認
			「昨日の課題、提出してくれてありがとう！みんなの3要素、全部確認した」
			「今日のワークで使うから手元に出しておいて。PC準備できてる方は「準備OK」と送ってください」
			→ 提出前提で入る。「書いてみてどうだった？」と感想を促してもOK
		A-2〜A-4: アウトプット紹介スライド（3〜4件）
			スキル / ①誰 / ②現状 / ③理想 + なぜ商品になるかコメント
			「今日のワークでさらに深まる」という予告として紹介する
		A-5: ブリッジスライド
			「昨日: 商品の「芯」ができた / 今日: AIで30秒で"形"にする」
	[BLOCK B] ロードマップ + 振り返り + Today's Goal（5〜6枚）
		B-1: 5Daysロードマップ（Day3強調）
			Day1: "売れるものがない"の思い込みを壊した ✅
			Day2: 設計図の全体像が入った ✅
			Day3: AIで設計図を動かす ← 今ここ
			Day4: 重要な話がある 🔒
			Day5: 総まとめ＋次の一手 🔒
			デザインポイント: Day4を🔒で表示して謎を残す
		B-2: Day1・Day2の振り返り（フロー型）
			Day1: 「資産はある（スキル × フォロワー）」
			Day2: 「設計図がない人は売れない / 設計図の全体構造を知った」
			Day3（今日）: 「設計図を、自分のスキルで動かす」
		B-3: 振り返り — Day1の核心
			"売れるものがない"は幻想だ。
			スキル × フォロワー = 既に「資産」
		B-4: 振り返り — Day2の核心
			❌ 設計図なし: とりあえず発信 → 売れない
			✅ 設計図あり: 集客 → 教育 → 販売（全工程を設計）
		B-5: Today's Merit
			① 自分の商品コンセプトが30秒で形になる
			② LP文章のドラフトが出来上がる
			③ "分かった"が"体験した"という実感に変わる
			＋ 今日だけのお楽しみ（後で明かします）
	[BLOCK C] 解説 — 設計図を自分に当てはめる思考（3〜4枚）
		C-1: 「当てはめる」とは何か（映像制作スキルの具体例）
			スキル: 映像制作 → 設計図に当てはめると…
			集客: Xで「映像初心者が必ずやる3つの失敗」を投稿 → CLへ誘導
			教育: 映像ビジネス5Days ワーク: 自分のスキルを商品化する演習
			販売: 「映像制作スキルで収益化する個別サポートプラン」案内
		C-2: AIを「設計に使う」vs「作業に使う」
			AIを作業に使う（多くの人）: 投稿ネタ生成・記事下書き・返信自動化 → 差がつかない
			AIを設計に使う（今日やること）: 商品コンセプト設計・LP構成設計・セールスフロー設計 → 再現性ある仕組み
		C-3: 今日の流れ確認
			Step1: 商品コンセプトをAIで生成（プロンプトコピペ）
			Step2: LP文章ドラフトをAIで生成
			Step3: 他の参加者の商品に値段をつけ合う
			Step4: ??? （後で明かします）
	[BLOCK D] AIワーク① — 商品コンセプト生成（3〜4枚）★
		D-1: ワーク①準備スライド
			✅ 昨日書いた3要素を手元に用意
			✅ ChatGPT or Claude を開く
			✅ 次のスライドのプロンプトをコピー
		D-2: プロンプトスライド（★最重要・大きく表示・コピペ用）
			あなたはオンライン講座のコンセプト設計の専門家です。
			【スキル】: ___
			【ターゲット（誰）】: ___
			【現状（Beforeの状態）】: ___
			【理想（Afterの状態）】: ___
			出力: ①商品タイトル3案 ②ターゲット像詳細 ③核心メッセージ ④LP構成骨格
		D-3: ワーク実施中スライド
			「やってみてください！ ⏱目安: 30秒〜1分」
			「出てきたら「出た！」とチャットに送ってください」
		D-4: ワーク①完了後まとめ
			AIが出してくれるのは「ドラフト」
			ゼロから3時間 → AIで30秒。これが「設計の自動化」
	[BLOCK E] AIワーク② — LP文章ドラフト生成（3枚）★
		E-1: ワーク②プロンプトスライド（★コピペ用・大きく）
			先ほどのLP構成1番目の見出しについてLP用文章を書いてください
			条件: ターゲット・トーン（親しみやすいが信頼感）・200〜300文字
		E-2: ワーク実施中スライド
			「1つ目のセクションだけでOK ⏱目安: 30秒〜1分」
			「「使える！」or「要修正」をチャットに送ってください」
		E-3: ワーク②完了後まとめ
			「要修正」は当然。それでいい。
			全セクション埋めれば → 今日中にLP骨格が完成
	[BLOCK F] 相互値付けチャレンジ（2〜3枚）
		F-1: チャレンジ案内
			オプチャに投稿: ①商品タイトル ②ターゲット一行 ③あなたが考える価格
			他の人の商品を見て「○円なら即買い」とコメント
		F-2: 実施中スライド（3〜5分）
			今すぐオプチャを開いて投稿・コメント
		F-3: 体験のまとめ
			自分のスキルに、他の参加者がリアルで値段をつけてくれた。
			需要は、ある。設計図を整えれば、それが売上になる。
	[BLOCK G] オープンループ（★Day3のクライマックス）（2枚）
		G-1: 全工程図スライド（★5〜10秒だけ表示・圧倒感演出）

		G-2: 引きスライド
			「これ、全部一人でできますか。」

	[BLOCK I] 付録スライド
		Q&Aスライド（想定Q&A一覧）

	インタラクションパターン（Day3）
		冒頭: PC確認 →「準備OK」チャット
		アウトプット紹介: 代弁型承認 → 商品案を褒めながらコメント
		振り返り: フロー確認 → 「Day1→Day2でどう変わった？」
		AIワーク中: 進行確認 → 「出た！」チャット促進
		ワーク後: 感想収集 → 「使える！」or「要修正」投票
		相互値付け: オプチャ交流 → リアルタイムで読み上げ
		オープンループ: 問いかけ → 「全部一人でできる？」無回答で終わる
	スライド制作上の注意点
		AIプロンプトスライドは大きく — コピペが最優先
		ワーク中スライドはシンプルに
		全工程図は情報量多め — 圧倒感を出すためにあえて詰め込む
		Day4の告知スライドは「謎」を残す — 内容は書かない
