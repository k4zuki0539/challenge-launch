## 重要な依存関係

```
Phase 1 確定 → Phase 2 着手可能
Phase 2 確定 → Phase 3 着手可能（問題提起1〜3が台本の骨格）
Phase 3 完了 → Phase 4 着手可能（教育の流れが決まってからワークを入れる）
Phase 5 完了 → Phase 8 着手可能（導線がないと集客できない）
Phase 6,7 完了 → Phase 9 着手可能
```

## リスクポイント

| リスク | 対策 |
|--------|------|
| Phase 1〜2 が長引く | コンセプトは完璧を求めず、70%で走り出す。台本制作中に磨く |
| Phase 3 の台本が重い | Day1から順に作り、前日の学びを次に反映。並行でスライド制作しない |
| 集客が伸びない | Phase 8開始時点で目標人数を設定。1週間で未達なら追加施策を即判断 |
| バックエンド商品が未完成 | Phase 5 開始前（5/15）までに商品ページ・決済導線を最低限用意 |
