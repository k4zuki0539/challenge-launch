## 1. このローンチの背景

### なぜ今これをやるのか

スキルはある。月5万くらいは稼げている。クライアントワークの実績だってある。

なのに、毎日が忙しいまま終わる。案件をこなすだけで精一杯で、やってみたいことは全部「また今度」になっている。商品を作りたいという気持ちはある。でも何から手をつければいいかわからないし、そもそも「自分のスキルが商品になるのか」という自信が持てない。

スキルがある。稼いでいる。なのに、時間も自信も足りない。

この"足りない"を埋めるために、1000万のローンチを再現したローンチ手法を公開する。時間に追われる生活を抜け出して、好きなことに時間を使えるようになるための5日間。

**前回CLの失敗教訓（ここが起点）：**
- 「アウトプット（何で稼ぐか）が不明確だと成約しない」→ Day1から商品化の体験を入れる
- 「比較検討ファネルまで到達できなかった」→ ペルソナを絞り込んで刺さる設計にする
- 「スマホ勢・低リテラシー層を前提にしていなかった」→ 全設計の前提を見直す

## 2. このローンチの目的

## 3. 期待される効果

**参加者側の変化：**

「ローンチって、特別な才能やセンスがいるものだと思っていた。でも違った。やり方を知っているかどうかだけだった。」

5日間を通じて、参加者はローンチの流れを、初めて最初から最後まで通して理解する。人を集めて、興味を持ってもらって、買ってもらうまでが、どう繋がっているのか。何をどの順番でやればいいのか。それがわかった瞬間、「自分にもできるかもしれない」という小さくて確かな気づきが生まれる。

さらに、「ここが大変そう」「時間がかかりそう」と腰が重かった作業が、AIを使うことで一瞬で片付く体験をする。これが「やり方を知っている人間」と「知らない人間」の差だと、頭ではなく手を動かして理解する。

**ビジネス側の効果：**
- わどAIスクールへの面談獲得
- 前回CLで課題だった「比較検討ファネルへの到達」を解消
- テイカー排除（Day3実践・Day4覚悟確認）によるコミュニティ品質向上

## 4. ターゲット定義

### ターゲットの本質

**スキルも収益手段もある。でもローンチができない。それだけで止まっている人。**

何かしら稼ぐ手段は持っている。クライアントワーク・AI副業・既存のコンテンツ販売、形は問わない。でも「自分の商品を持って、ローンチで一気に稼ぐ」という次のステップに進めていない。やり方を知らないか、時間がないか、自信がないか。理由はこの3つだけ。

| 項目 | 詳細 |
|------|------|
| 年齢・属性 | 35〜39歳、男性、会社員 |
| スキル | 映像制作・IT・デザイン・ライター等。本業で培ったスキルあり |
| 収益状況 | 副業で月数万の収益あり、または収益化を何度か試みた経験あり |
| 家族 | 子ども1〜2人。住宅ローンあり。教育費・物価高への不安がある |
| 悩み | 案件をこなすだけで時間が消える。商品を作る余裕がない。ローンチのやり方がわからない。**→ つまり：** 帰って飯食って風呂入ったら22時。そこから「さあ商品作るか」とはならない。土日も家族の時間や溜まった案件で終わる。「やろう」と思っているのに、やれる日が永遠に来ない |
| 理想 | 本業＋自分の商品収益で月50万。子どもの教育費の不安をなくしたい。**→ つまり：** お金の不安が消えることで、子どもの「やりたい」に即答できる親になれる。選択肢をお金で諦めさせない、という具体的なゴール |
| ブレーキ | 「ローンチは難しそう」「自分にできるか自信がない」「時間が作れない」。**→ つまり：** 難しいという思い込みと時間の言い訳が合わさって、着手のハードルが無限に上がり続けている |

「動画編集講座を大量に作ってコンテンツとして販売をしたいです。ですが、どうやったら良いのかよく分からないです」

## 5. 悩みの深掘り（3ペルソナ共通）

### レイヤー1：表面の悩み（本人が言語化できている）

**「時間がない。案件をこなすだけで1日が終わる」**
→ それってつまり？
→ 休憩が仕事の合間にあるのではなく、仕事が休憩の合間にある状態。疲れを取るための時間まで削られていて、新しいことを考える気力が残っていない。

**「もっと稼ぎたい。でも案件を増やすと体が壊れる」**
→ それってつまり？
→ 体を壊す覚悟がなければ収入を増やせない、という状況に自分からはまり込んでしまっている。努力すれば報われる、ではなく、努力すれば壊れる、という天井にぶつかっている。

**「発信しているのにフォロワーが増えても収入に繋がらない」**
→ それってつまり？
→ フォロワーはいるのに、そこから売上を作る方法を持っていない。宝の持ち腐れの状態が続いている。発信の努力が全て無駄になっている感覚。

---

### レイヤー2：本質の悩み（気づき始めているが言語化できていない）

**「商品を作りたい。でも何から手をつければいいかわからない」**
→ それってつまり？
→ ゴールは見えているのにスタートラインが見えない。何をどの順番でやればいいかがわからないから、考えるほど頭がこんがらがって動けなくなる。「わからない」ではなく「どこから始めるかがわからない」という詰まり方。

**「自分のスキルが商品になるとは思えない」**
→ それってつまり？
→ 毎日やっていることだから自分には当たり前に感じる。でも外から見ると希少価値がある。自分だけが自分のスキルを過小評価している。市場は買いたがっているのに、売り手が「こんなもの売れない」と思い込んでいる。

**「ローンチや1000万の話は、自分とは別次元の話」**
→ それってつまり？
→ 成功している人を「自分とは違う生き物」として認識している。だから再現しようという発想が最初から生まれない。「あの人だからできた」で思考が止まる。

---

### レイヤー3：感情の痛み（言いたくない・恥ずかしい）

**「スキルはある。でも商品にするほど自信がない。否定されるのが怖い」**
→ それってつまり？
→ 動かない理由は「準備不足」ではなく「失敗の確定が怖い」から。今は「いつかやれる」という可能性が残っている。でも実際にやって失敗すると、その可能性が消える。可能性を守るために、あえて動かないでいる。

**「クライアントワークはやめるのも怖い。続けるのも嫌」**
→ それってつまり？
→ このままでいるのも嫌。でも踏み出すのも怖い。どちらを選んでも苦しい、という八方塞がりの感覚。

**「やりたいことはある。でも"そんな余裕はない"と自分でブレーキをかける」**
→ それってつまり？
→ 夢を守るために、あえて挑戦しない。挑戦して失敗するより、挑戦しない言い訳を作り続ける方が心理的に楽。これが「いつかやろう」の正体。

---

### レイヤー4：時代の不安（じわじわ大きくなっている恐怖）

**「AIが自分の仕事（ライター・デザイン・動画編集等）を奪いつつある」**
→ それってつまり？
→ AIのせいで、自分が頑張っても稼ぎが下がっていく。自分の努力と関係ないところで、じわじわ追い詰められている。

**「AIを使いこなせる人は稼いでいる。自分は乗り遅れているかもしれない」**
→ それってつまり？
→ 今のうちに動かないとまずいという焦り。でも何をすれば正解かわからない。焦りだけが先行して、行動できない状態。

**「情報が多すぎて、何が本物かわからない」**
→ それってつまり？
→ 信頼できる情報源がない。だからどこにも踏み込めない。調べるほど混乱して、情報収集自体が不安の原因になっている。

## 6. 悩みを放置するとどうなるか（末路）

### 1年後のシーン

相変わらず案件をこなしている。収入はほとんど変わっていない。むしろ、単価が少し下がってきた。AIの普及で「同じ仕事をもっと安くできる人（or AI）」が増えたから。

「商品を作ろう」と思い続けてもう2年が経つ。メモ帳にはアイデアが20個書いてある。でも一つも形になっていない。夜、Xで「自分の商品で稼いでいる人」の投稿を見て、羨ましいと思い、「自分には無理かな」と閉じる。この繰り返し。

### 具体的な末路

**「時間の罠から抜け出せない」**
→ それってつまり？
→ 働かなければ収入がゼロになる恐怖が、いつまでも「時間を売る」ことを強制する。体が動く限りは稼げる。でも体が動かなくなった瞬間、収入もゼロになる。その状態から一生抜け出せない。

**「収入の上限が体力で決まる」**
→ それってつまり？
→ 頑張れば頑張るほど体が壊れ、休めば休むほど収入が下がる。頑張るほど疲れるだけで、何も積み上がっていかない。

**「AI時代に作業者のまま置いていかれる」**
→ それってつまり？
→ 自分のスキルがAIに代替されていく速度は、努力では止められない。単価が下がり続け、同じ収入を維持するために作業量を増やす悪循環に入る。

**「やりたいことが"昔の夢"になる」**
→ それってつまり？
→ 「いつかやろう」と言い続けた結果、そのアイデアが時代遅れになる。または、気力が尽きて「もうやらなくていいか」という諦めに変わる。夢が消えるのではなく、夢を語る気力が消える。

**「"自分には無理"という思い込みが固まる」**
→ それってつまり？
→ 動かない時間が長くなるほど、「やっぱり自分には無理だった」という証拠が積み上がっていく。最初は根拠のない思い込みだったのに、年数が経つと「事実」に変わっていく。

## 7. 悩みを解決したらどうなるか（理想状態）

### 5日間のCL終了直後（短期変化）

「5日間で、ローンチってこういうことか、と初めてわかった気がした。全体の流れが見えた瞬間、"難しそう"という感覚が消えた。難しいんじゃなくて、知らなかっただけだった。

今まで『自分には無理』と思っていたのが、なぜかと聞かれると答えられない。設計図はある。AIもある。スキルもある。あとは環境だけ、という感覚になっている。」

→ それってつまり？
→ 長年「謎」だったものが解けた体験。「自分には無理」という思い込みの根拠を、自分で崩してしまった状態。これ以降、同じ言い訳が使えなくなる。

### わどAIスクール入会後3〜6ヶ月（中期変化）

「初めてのローンチで収益が出た。クライアントワーク数件分の収入が、自分の商品1回のローンチで出た。

一番変わったのは時間の使い方。以前は空き時間に案件をこなしていた。今は空き時間に自分のコンテンツを育てている。同じ時間を使っているのに、積み上がっていく感覚がまったく違う。」

→ それってつまり？
→ 同じ時間・同じ労力でも、自分の商品を持つと何倍もの収益になることを体で理解した。クライアントワークは終わればリセットされる。自分のコンテンツは積み上がっていく。時間の使い方の意味が根本から変わった。

### 長期的な理想状態

**「時間の自由を手に入れる」**
→ それってつまり？
→ 月曜日の朝、アラームなしで起きて、その日やることを自分で決める。誰の許可もいらない。子どもの行事に行けるか会社に伺いを立てる必要がない。人生のハンドルが自分の手に戻っている。

**「収入の天井が消える」**
→ それってつまり？
→ 「働いた時間 × 時給」という公式から解放される。寝ている間も、旅行中も、収益が積み上がる。体力が稼ぎの上限を決める生き方が終わる。

**「自信が実績に変わる」**
→ それってつまり？
→ 「自分のスキルが商品になった」「誰かが買ってくれた」という事実が、根拠のある自信に変わる。以前は"思い込み"で止まっていたのに、今は"実績"が背中を押す。

**「AI時代に乗る側になる」**
→ それってつまり？
→ 頼まれた仕事をこなす側から、自分で仕事を作る側へ。AIに仕事を奪われる側ではなく、AIを使って稼ぐ側に移っている。

**「お金と時間の制約から来る諦めがなくなる」**
→ それってつまり？
→ 「行きたいけどお金が…」「やりたいけど時間が…」という言い訳が消える。家族旅行・子どもの習い事・自分の学び、全部「どれにしようか」という選択になる。諦めが選択に変わる。

## 8. ターゲットへの訴求で注意すること（前回反省から）

1. **「稼げていない人」向けではない**: 月5万稼げている人が対象。「ゼロから稼ぐ」の文脈で訴求すると刺さらない
2. **時間の問題を最初に共感する**: 「忙しい」「時間がない」の共感から入る。稼ぎたい欲より先に時間の悩みがある
3. **自信のなさをケアする**: 「自分には無理かも」という自己否定が行動の最大のブレーキ。Day1でここを崩す
4. **スマホ勢前提で設計**: 複雑な操作・PC必須の設計は離脱を生む
5. **金額提示がないと比較検討に入れない**: 価格を早い段階で示すことが「比較検討ファネル」への到達に必要
6. **本気度を可視化する仕掛け**: Day3の実践ワークで課題提出させ、テイカーと本気勢を分ける
